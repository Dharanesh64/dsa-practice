# DSA Important Questions & Patterns

This repository contains a collection of important Data Structures and Algorithms questions along with common problem-solving patterns frequently asked in maang

The goal of this repository is to organize coding problems based on patterns such as arrays, sorting, hashing, sliding window, and dynamic programming. These patterns help in understanding how different problems can be solved using similar approaches.

This repository is mainly created for last minute revision of important problems for software engineering roles.

## Topics Covered

* Arrays
* string
* Sorting Algorithms
* Hashing
* Stack & Queue
* Sliding Window
* Linked List
* Trees & Graph
* Dynamic Programming
* SQL Interview Queries

## Purpose

* Practice important interview questions
* Understand common problem-solving patterns
* Improve coding and algorithmic thinking
